Transparencia de datos

Politica de privacidad

Esta politica explica que datos tratamos, para que los usamos y como puedes ejercer tus derechos.

Ultima actualizacion: 25 de febrero de 2026

1. Datos que recopilamos

Recopilamos datos de cuenta (nombre, correo, telefono), datos comerciales (catalogo, precios, stock), y datos operativos de uso de la plataforma.

2. Finalidades del tratamiento

Usamos la informacion para autenticar usuarios, operar cotizaciones, mostrar catalogos, mejorar funciones y entregar soporte tecnico.

3. Base de legitimacion

Tratamos datos por ejecucion del contrato de servicio, cumplimiento normativo y, cuando corresponda, consentimiento del usuario.

4. Conservacion y seguridad

Conservamos datos solo por el tiempo necesario para operar el servicio y cumplir obligaciones legales. Aplicamos controles de acceso, trazabilidad y medidas de seguridad razonables.

5. Comparticion de datos

No vendemos datos personales. Compartimos informacion con proveedores de infraestructura y herramientas necesarias para operar, bajo obligaciones de confidencialidad.

6. Derechos del titular

Puedes solicitar acceso, rectificacion o eliminacion de tus datos, cuando aplique legalmente, escribiendo a privacidad@construcomparador.cl.

7. Consultas

Si necesitas apoyo, visita contacto o revisa preguntas frecuentes.
